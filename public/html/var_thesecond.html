VarDrugPub_v2 :   Variant-Gene-Drug Relation Database The Second

  Summary: VarDrugPub_v2 is a database that contains genomic variants and pharmacogenomic information. There are numerous research studies published in PubMed about the association/relationship between drug responses and somatic mutations. We use a deep learning-based text mining technique to identify mutation-gene-drug relations in all the PubMed articles. You can use VarDrugPub at http://vardrugpub.korea.ac.kr. Yes, we know that the description is exactly the same as the VarDrugPub_v1. Currently our team of highly trained monkeys are trying to append more detailed decription. Please wait for more time.
